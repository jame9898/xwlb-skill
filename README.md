# 新闻联播文字版获取工具

获取每日央视新闻联播的文字版内容，支持查看新闻简介和详细内容。

## 功能特点

- **智能获取**: 支持查询任意历史日期的新闻联播
- **新闻简介**: 快速浏览当天所有新闻标题
- **详细内容**: 查看每条新闻的完整文字内容
- **双格式输出**: 支持 TXT 和 MD 两种格式
- **并发处理**: 5线程并发获取，速度快
- **进度反馈**: 实时显示获取进度
- **智能重试**: 请求失败自动重试，带延迟策略
- **文件防覆盖**: 文件名包含运行日期，避免同一天多次运行覆盖
- **友好提示**: 获取失败时给出明确提示
- **数据来源**: 央视网官方 tv.cctv.com

## 安装

```bash
pip install requests
```

## 使用方法

### 1. 获取新闻简介

```bash
python get_xwlb.py today              # 查看今天的新闻简介
python get_xwlb.py yesterday          # 查看昨天的新闻简介
python get_xwlb.py 2025-09-09         # 查看指定日期的新闻简介
python get_xwlb.py 20250909           # 支持YYYYMMDD格式
```

### 2. 获取详细内容

```bash
python get_xwlb.py today 1            # 获取今天所有新闻详情（TXT格式）
python get_xwlb.py today 1 --md       # 获取今天所有新闻详情（MD格式）
python get_xwlb.py 2025-09-09 1       # 获取指定日期详情（TXT格式）
python get_xwlb.py 2025-09-09 1 --md  # 获取指定日期详情（MD格式）
```

## 文件结构

```
新闻联播/
├── News_Summary/                    # 简介文件夹
│   └── YYYYMMDD_summary.txt/md      # 简介文件
├── News_Detail/                     # 详情文件夹
│   └── YYYYMMDD_运行日期_detail.txt/md  # 详情文件（带运行日期防覆盖）
├── get_xwlb.py
├── README.md
└── requirements.txt
```

## 输出格式

### TXT 格式

**简介文件**：
```
==================================================
XXXX年XX月XX日 新闻联播简介如下：
==================================================
第一条，这是第一条新闻标题
第二条，这是第二条新闻标题
第三条，这是第三条新闻标题
...
==================================================
（来源：XXXX年XX月XX日的新闻联播 - 央视网新闻联播主页 - https://tv.cctv.com/lm/xwlb/index.shtml）
==================================================
```

**详情文件**：
```
==================================================
第一条：这是第一条新闻标题
==================================================

　　这是第一条新闻的详细内容...

==================================================
第二条：这是第二条新闻标题
==================================================

　　这是第二条新闻的详细内容...
```

### MD 格式

**简介文件**：
```markdown
# XXXX年XX月XX日 新闻联播

> 来源：央视网新闻联播主页 - https://tv.cctv.com/lm/xwlb/index.shtml

第一条：这是第一条新闻标题

第二条：这是第二条新闻标题

第三条：这是第三条新闻标题
```

**详情文件**：
```markdown
# XXXX年XX月XX日 新闻联播

> 来源：央视网新闻联播主页 - https://tv.cctv.com/lm/xwlb/index.shtml

## 第一条：这是第一条新闻标题

　　这是第一条新闻的详细内容...

## 第二条：这是第二条新闻标题

　　这是第二条新闻的详细内容...
```

### 联播快讯格式

**TXT 格式**：
```
==================================================
第X条：国内联播快讯
==================================================

【快讯标题一】
　　这是快讯一的详细内容...

【快讯标题二】
　　这是快讯二的详细内容...
```

**MD 格式**：
```markdown
## 第X条：国内联播快讯

***快讯标题一***
　　这是快讯一的详细内容...

***快讯标题二***
　　这是快讯二的详细内容...
```

## 技术特点

- **并发请求**: 使用 ThreadPoolExecutor 实现5线程并发获取
- **连接复用**: 使用 requests.Session 复用HTTP连接
- **智能重试**: 请求失败自动重试，带1秒延迟
- **进度反馈**: 实时显示 `[X/Y] 已获取: 新闻标题...`
- **文件防覆盖**: 文件名格式 `目标日期_运行日期_detail.txt`
- **错误处理**: 全部失败时提示用户，部分失败时跳过并提示

## 注意事项

- 需要能够访问央视网 tv.cctv.com
- 新闻联播每天19:00播出，发布时间可能因节目时长而延迟
- 建议不要频繁请求，以免对服务器造成压力
- 同一天多次运行会生成不同时间戳的文件，不会覆盖
